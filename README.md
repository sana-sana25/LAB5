# Converter Tabs App (LAB5)

##  Objectif

Développer une application Android avec deux onglets :

* Conversion de température (°C ↔ °F)
* Conversion de distance (Km ↔ Miles)

---

##  Fonctionnalités

- Interface avec onglets (TabLayout + ViewPager2)
- Conversion Température
- Conversion Distance
- Menu "Quitter"
- Boîte de confirmation avant fermeture

---

##  Technologies utilisées

* Java
* Android Studio
* Material Design
* ViewPager2
* Fragments

---

##  Structure du projet

* `MainActivity.java` : activité principale
* `ViewPagerAdapter.java` : gestion des fragments
* `TempFragment.java` : conversion température
* `DistanceFragment.java` : conversion distance

---

## Formules utilisées

###  Température

* Celsius → Fahrenheit :
  `F = (1.8 × C) + 32`

* Fahrenheit → Celsius :
  `C = (F − 32) / 1.8`

---

### Distance

* Km → Miles :
  `Miles = Km × 0.6214`

* Miles → Km :
  `Km = Miles / 0.6214`

---

##  Lancement

1. Ouvrir le projet avec Android Studio
2. Lancer l’émulateur
3. Cliquer sur Run ▶️
